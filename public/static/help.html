ヘルプセンター

よくある質問と詳しい使い方のガイドです

  1. トップページの「新規登録」ボタンをクリック
  2. メールアドレス、名前、パスワードを入力
  3. 利用規約に同意して「登録」をクリック
  4. 登録したメールアドレスに確認メールが送信されます
  1. ログインページの「パスワードを忘れた方」をクリック
  2. 登録したメールアドレスを入力
  3. 送信されたリセットリンクから新しいパスワードを設定
  1. ダッシュボードの「新規案件」ボタンをクリック
  2. 物件名、住所、面積、価格などの基本情報を入力
  3. ステータス（調査中、交渉中など）を選択
  4. 「保存」ボタンで案件を作成
  5. 作成後、詳細情報やファイルを追加できます

各案件にはチャット機能が付いています：

  • 案件詳細ページの「チャット」タブを開く
  • メッセージを入力して送信
  • @ユーザー名で特定のメンバーにメンション
  • ファイルアイコンからファイルを添付可能
  • 過去のメッセージは検索機能で簡単に見つけられます

2つの方法でファイルをアップロードできます：

  1. ファイル管理から: 案件詳細の「ファイル」タブで「アップロード」ボタンをクリック
  2. チャットから: メッセージ送信時にファイルを添付

対応形式: PDF, Word, Excel, 画像ファイル（最大10MB）

  1. 右上のユーザーメニューから「設定」を選択
  2. 「通知設定」タブを開く
  3. メール通知やプッシュ通知の項目をオン/オフ
  4. 「保存」ボタンで設定を更新

以下の点を確認してください：

  • メールアドレスとパスワードが正しいか確認
  • Caps Lockがオンになっていないか確認
  • ブラウザのキャッシュをクリアしてみる
  • パスワードリセット機能を試す
  • それでも解決しない場合は管理者に連絡

以下を確認してください：

  • ファイルサイズが10MB以下であるか
  • 対応ファイル形式（PDF, Word, Excel, 画像）であるか
  • インターネット接続が安定しているか
  • ブラウザを最新バージョンに更新
  1. ページを再読み込み（F5キー）
  2. ブラウザのキャッシュとCookieをクリア
  3. 別のブラウザで試してみる（Chrome, Firefox, Edge推奨）
  4. ブラウザを最新バージョンに更新
