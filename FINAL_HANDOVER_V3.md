## 🌐 独自ドメイン設定

**完全ガイド**: `CUSTOM_DOMAIN_SETUP.md`

### クイックスタート

#### ステップ1: ドメイン取得
```
推奨: Cloudflare Registrar
価格: 約$10-$15/年
```

#### ステップ2: DNS設定
```
Type: CNAME
Name: app（またはサブドメイン名）
Target: real-estate-200units-v2.pages.dev
Proxy: ✅ Proxied
```

#### ステップ3: Cloudflare Pagesにドメイン追加
```bash
npx wrangler pages domain add app.yourcompany.com \
  --project-name real-estate-200units-v2
```

#### ステップ4: SSL/TLS設定
```
モード: Full (strict)
Always Use HTTPS: ON
```
