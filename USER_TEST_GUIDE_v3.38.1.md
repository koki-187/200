## ❓ よくある質問

### Q1: コンソールログが表示されない
**A**: F12キーで開発者ツールを開き、「Console」タブを選択してください。ブラウザによってはF12キーではなく、以下のショートカットです：
- Windows/Linux: F12 または Ctrl + Shift + I
- Mac: Option + Command + I

### Q2: favicon.ico の500エラーが表示される
**A**: これは無視して問題ありません。アプリケーションの動作には影響しません。

### Q3: Tailwind CSS の警告が表示される
**A**: これも無視して問題ありません。本番環境での最適化推奨の警告ですが、動作には影響しません。

### Q4: ログインしてもダッシュボードに移動しない
**A**: 以下を確認してください：
1. ブラウザのJavaScriptが有効になっているか
2. コンソールにエラーメッセージが表示されていないか
3. ネットワークタブで `/api/auth/login` リクエストが成功（200 OK）しているか
