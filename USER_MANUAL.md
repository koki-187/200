## システム概要

### システムの目的
200戸マンション用地取得案件を効率的に管理するための不動産仲介業者向けシステムです。

### 主な機能
- **案件管理**: 土地仕入れ案件の登録・追跡・管理
- **メッセージング**: 案件に関するコミュニケーション
- **ファイル管理**: 契約書類、図面などのアップロード・共有
- **OCR機能**: 書類の自動読み取り（OpenAI GPT-4 Vision）
- **通知機能**: メール・プッシュ通知でリアルタイム更新
- **KPIダッシュボード**: 案件進捗状況の可視化
- **バックアップ**: 定期的なデータバックアップ

### アクセス URL
**本番環境**: https://real-estate-200units-v2.pages.dev

## ログイン方法

### 1. ログインページにアクセス
ブラウザで https://real-estate-200units-v2.pages.dev を開きます。

### 2. 認証情報を入力
- **メールアドレス**: 管理者から提供されたメールアドレス
- **パスワード**: 初回ログイン時に設定したパスワード

### 3. ログインボタンをクリック
入力内容を確認し、「ログイン」ボタンをクリックします。

### 初回ログイン時のパスワード変更
セキュリティのため、初回ログイン後は必ずパスワードを変更してください。

## 画面構成

### ダッシュボード
- **KPI概要**: 案件数、進捗状況、成約率などの主要指標
- **最近の案件**: 更新された案件のリスト
- **通知センター**: 未読通知の表示

### 案件一覧画面
- **検索・フィルター**: ステータス、地域、優先度で絞り込み
- **ソート機能**: 作成日時、更新日時、優先度で並び替え
- **一括操作**: 複数案件の一括編集・削除

### 案件詳細画面
- **基本情報**: 案件名、所在地、面積、価格など
- **進捗管理**: ステータス、期限、担当者
- **ファイル管理**: 関連ファイルのアップロード・閲覧
- **メッセージ**: 案件に関するコミュニケーション履歴
- **アクティビティログ**: 変更履歴の確認

## 主要機能

### 1. 案件管理

#### 新規案件作成
1. ダッシュボードで「新規案件」ボタンをクリック
2. 必須項目を入力:
   - 案件名（タイトル）
   - 所在地
   - 買主・売主情報
   - 土地面積
   - 希望価格
3. 「作成」ボタンをクリック

#### 案件編集
1. 案件一覧または詳細画面で「編集」ボタンをクリック
2. 変更したい項目を修正
3. 「保存」ボタンをクリック

#### ステータス管理
案件は以下のステータスで管理されます:
- **NEW**: 新規案件
- **IN_REVIEW**: レビュー中
- **REPLIED**: 回答済み
- **CLOSED**: 終了

### 2. ファイル管理

#### ファイルアップロード
1. 案件詳細画面の「ファイル」タブを開く
2. 「ファイルをアップロード」ボタンをクリック
3. ファイルを選択（PDF, Word, Excel, 画像など）
4. アップロード完了を待つ

#### OCR機能の利用
1. 画像ファイル（PDF、JPEG、PNG）をアップロード
2. 「OCR実行」ボタンをクリック
3. 自動的にテキストが抽出され、案件情報に反映

### 3. メッセージング

#### メッセージ送信
1. 案件詳細画面の「メッセージ」タブを開く
2. メッセージを入力
3. 必要に応じてファイルを添付
4. 「送信」ボタンをクリック

#### @メンション機能
メッセージ内で `@ユーザー名` を入力すると、特定のユーザーに通知されます。

### 4. 通知設定

#### 通知の種類
- **メール通知**: 重要な更新をメールで受信
- **プッシュ通知**: ブラウザ通知でリアルタイム更新
- **SMS通知**: 緊急時のSMS通知（オプション）

#### 通知設定の変更
1. 設定画面を開く
2. 「通知設定」タブを選択
3. 受信したい通知の種類をチェック
4. 「保存」ボタンをクリック

### 5. フィードバック送信

#### バグ報告・機能要望
1. 画面右下のフィードバックボタンをクリック
2. 種類を選択:
   - **bug**: バグ報告
   - **feature**: 機能要望
   - **improvement**: 改善提案
   - **other**: その他
3. タイトルと詳細を入力
4. 優先度を選択（low, medium, high）
5. 「送信」ボタンをクリック

## トラブルシューティング

### ログインできない

**原因1: パスワードが間違っている**
- パスワードを再確認してください
- 管理者にパスワードリセットを依頼してください

**原因2: アカウントが無効化されている**
- 管理者に連絡して、アカウントの有効化を依頼してください

### ファイルがアップロードできない

**原因1: ファイルサイズが大きすぎる**
- ファイルサイズの上限: 10MB（設定により変更可能）
- ファイルを圧縮するか、分割してアップロードしてください

**原因2: サポートされていないファイル形式**
- 対応形式: PDF, Word, Excel, JPEG, PNG, GIF
- 他の形式の場合は、PDFに変換してアップロードしてください

### OCRが正しく動作しない

**原因1: 画像品質が低い**
- 高解像度のスキャン画像を使用してください
- 文字がはっきり読める画像を使用してください

**原因2: 手書き文字が多い**
- OCRは印刷された文字に最適化されています
- 手書き文字が多い場合は、手動入力をお願いします

### 通知が届かない

**原因1: 通知設定が無効**
- 設定画面で通知設定を確認してください
- 必要な通知をONにしてください

**原因2: ブラウザの通知が許可されていない**
- ブラウザの設定で、サイトからの通知を許可してください
- Chrome: 設定 > プライバシーとセキュリティ > サイトの設定 > 通知

### その他の問題

上記で解決しない場合は、以下の情報を添えて管理者にお問い合わせください:
1. 発生日時
2. 実施した操作
3. エラーメッセージ（あれば）
4. スクリーンショット（あれば）
